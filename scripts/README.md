# Scripts de utilidad

Este directorio concentra comandos repetibles:
- Creacion y activacion de entornos.
- Tareas de limpieza o formateo.
- Descarga de datasets ligeros para ejemplos.

Ejemplo de script en Windows (`setup.ps1`):
```powershell
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```
