# Ciencia de Datos con Python

Este directorio contiene una propuesta curricular completa para una ruta de ciencia de datos con Python, disenada con nivel progresivo de licenciatura, maestria profesional y especializacion aplicada.

El temario integra:

- fundamentos de computacion y programacion,
- calculo, algebra lineal, probabilidad, estadistica y optimizacion,
- bases de datos, visualizacion, limpieza y comunicacion,
- regresion, algoritmos de prediccion, reduccion de dimensionalidad y aprendizaje no supervisado,
- deep learning, NLP, grafos, recomendadores e IA generativa,
- data engineering, big data, cloud, MLOps y gobierno de datos,
- etica, privacidad, producto analitico y proyectos de industria.

## Criterios de diseno

1. **Orden universitario**: primero herramientas, luego matematicas, despues estadistica, datos, modelos y sistemas.
2. **Profundidad de maestria**: incluye causalidad, modelos bayesianos, sistemas distribuidos, MLOps, cloud, investigacion y capstone.
3. **Python como hilo conductor**: cada materia tiene practicas concretas en Python.
4. **Enfoque industrial en Mexico**: se consideran manufactura, electronica, automotriz, retail, fintech, logistica, salud, energia, gobierno y PyMEs.
5. **Portafolio progresivo**: cada bloque debe producir evidencias: notebooks, scripts, dashboards, APIs, reportes y proyectos reproducibles.

## Secuencia sugerida

- **Anio 1**: Python, terminal/Git, matematicas discretas, calculo, algebra lineal y probabilidad.
- **Anio 2**: estadistica, optimizacion, SQL, ETL, visualizacion, EDA, regresion y clasificacion.
- **Anio 3**: ensambles, validacion, dimensionalidad, clustering, series de tiempo, NLP y deep learning.
- **Anio 4**: bayesiano, causalidad, big data, ingenieria de datos, MLOps, cloud, etica, producto y proyecto integrador.
- **Maestria/especializacion**: profundizar en investigacion, sistemas a escala, IA generativa, grafos, recomendadores, geoespacial y capstone industrial.

## Resultado esperado del egresado

Una persona que complete esta ruta debe poder:

- formular problemas de datos con contexto de negocio o politica publica,
- limpiar, modelar y validar datos con rigor estadistico,
- construir modelos predictivos e interpretarlos,
- desplegar soluciones reproducibles y monitoreables,
- comunicar resultados a audiencias tecnicas y ejecutivas,
- trabajar con responsabilidad etica y legal,
- adaptar soluciones a sectores productivos relevantes en Mexico.
