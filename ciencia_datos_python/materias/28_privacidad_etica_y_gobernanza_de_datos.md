# Privacidad, Etica y Gobernanza de Datos

**Nivel:** Licenciatura avanzada / Maestria

**Bloque:** Industria, etica y capstone

**Proposito:** Formar criterio para usar datos y modelos con responsabilidad tecnica, legal y social.

**Prerrequisitos:** EDA, estadistica y nociones de ML.

## Temario por unidades

1. Privacidad, consentimiento y minimizacion de datos.
2. Sesgo, equidad, fairness y representacion.
3. Explicabilidad, auditabilidad y documentacion.
4. Riesgos de modelos en decisiones sensibles.
5. Gobernanza: catalogos, linaje, propietarios y calidad.
6. Seguridad de datos y manejo de secretos.
7. Marco mexicano: proteccion de datos personales y responsabilidades organizacionales.

## Practicas en Python

- Auditar sesgo por subgrupos.
- Crear model cards y datasheets.

## Conexion con industria en Mexico

Aplicar criterio en credito, seguros, empleo, educacion, salud, seguridad y sector publico.

## Evaluacion sugerida

Auditoria de un modelo o dataset con recomendaciones.

## Evidencias de portafolio

- Repositorio limpio con README, ambiente reproducible y datos documentados.
- Notebook o script principal con resultados verificables.
- Reporte tecnico breve y presentacion ejecutiva.
