# Capstone Industrial de Maestria

**Nivel:** Maestria profesional

**Bloque:** Industria, etica y capstone

**Proposito:** Resolver un problema abierto con profundidad tecnica y valor practico para una organizacion.

**Prerrequisitos:** Cursos nucleares de maestria.

## Temario por unidades

1. Contrato de proyecto: alcance, datos, entregables y riesgos.
2. Arquitectura de solucion y criterios de exito.
3. Pipeline de datos y versionado.
4. Modelado, validacion y experimentos.
5. MLOps o producto analitico.
6. Etica, privacidad, costos y mantenimiento.
7. Reporte ejecutivo, reporte tecnico y transferencia.

## Practicas en Python

- Construir una solucion completa desplegable o reproducible.
- Preparar documentacion para mantenimiento.

## Conexion con industria en Mexico

Priorizar proyectos con manufactura, fintech, retail, logistica, salud, energia, gobierno o PyMEs.

## Evaluacion sugerida

Demo, defensa tecnica, impacto medible y documentacion de continuidad.

## Evidencias de portafolio

- Repositorio limpio con README, ambiente reproducible y datos documentados.
- Notebook o script principal con resultados verificables.
- Reporte tecnico breve y presentacion ejecutiva.
