# Calculo para Ciencia de Datos

**Nivel:** Licenciatura, semestre 2

**Bloque:** Fundamentos matematicos

**Proposito:** Entender cambio, aproximacion y optimizacion como base de aprendizaje automatico.

**Prerrequisitos:** Algebra, funciones y trigonometria.

## Temario por unidades

1. Funciones, limites, continuidad y aproximacion local.
2. Derivadas, gradiente, regla de la cadena y derivacion implicita.
3. Optimizacion de una variable y varias variables.
4. Integrales, area bajo la curva y valor esperado.
5. Calculo multivariable: gradiente, Jacobiano y Hessiano.
6. Series de Taylor y aproximaciones numericas.
7. Interpretacion de perdida, gradiente descendente y sensibilidad.

## Practicas en Python

- Visualizar funciones, derivadas e integrales numericas.
- Implementar gradiente descendente para una regresion lineal simple.

## Conexion con industria en Mexico

Modelar costos, demanda, energia, inventarios y rendimiento de procesos.

## Evaluacion sugerida

Problemas teoricos, notebooks de visualizacion y practica de optimizacion.

## Evidencias de portafolio

- Repositorio limpio con README, ambiente reproducible y datos documentados.
- Notebook o script principal con resultados verificables.
- Reporte tecnico breve y presentacion ejecutiva.
