# Ingenieria de Datos y Data Warehousing

**Nivel:** Maestria profesional

**Bloque:** Ingenieria y produccion

**Proposito:** Disenar datos confiables para analitica, BI y machine learning en organizaciones reales.

**Prerrequisitos:** SQL, ETL y bases de datos.

## Temario por unidades

1. Modelado dimensional: hechos, dimensiones y granularidad.
2. Data warehouse, data lake y lakehouse.
3. ELT con dbt y pruebas de datos.
4. Orquestacion con Airflow o Prefect.
5. Data contracts, linaje y observabilidad.
6. Calidad, master data y catalogacion.
7. Serving layer para BI y ML.

## Practicas en Python

- Construir transformaciones reproducibles.
- Crear tests de datos y documentacion automatica.

## Conexion con industria en Mexico

Arquitecturas para cadenas comerciales, bancos, aseguradoras, manufactura y sector publico.

## Evaluacion sugerida

Data mart funcional con documentacion, pruebas y dashboard.

## Evidencias de portafolio

- Repositorio limpio con README, ambiente reproducible y datos documentados.
- Notebook o script principal con resultados verificables.
- Reporte tecnico breve y presentacion ejecutiva.
