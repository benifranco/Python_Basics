# Programacion Python para Ciencia de Datos

**Nivel:** Licenciatura, semestre 1

**Bloque:** Herramientas computacionales

**Proposito:** Construir una base solida en Python para resolver problemas con datos antes de usar librerias complejas.

**Prerrequisitos:** Ninguno.

## Temario por unidades

1. Tipos de datos, variables, expresiones y conversiones.
2. Control de flujo: condicionales, ciclos y excepciones.
3. Funciones, parametros, retorno, alcance y documentacion.
4. Colecciones: listas, tuplas, diccionarios, conjuntos y comprensiones.
5. Archivos, rutas, CSV, JSON y serializacion basica.
6. Introduccion a NumPy, pandas y notebooks.
7. Depuracion, lectura de errores y pruebas pequenas.

## Practicas en Python

- Crear scripts reproducibles desde la terminal.
- Leer archivos CSV y JSON con pandas.
- Resolver ejercicios de limpieza y transformacion sin modelos.

## Conexion con industria en Mexico

Usar datos publicos sencillos de INEGI o datos simulados de comercio, empleo y educacion.

## Evaluacion sugerida

Tareas semanales, mini proyecto de limpieza y examen practico con archivos reales.

## Evidencias de portafolio

- Repositorio limpio con README, ambiente reproducible y datos documentados.
- Notebook o script principal con resultados verificables.
- Reporte tecnico breve y presentacion ejecutiva.
