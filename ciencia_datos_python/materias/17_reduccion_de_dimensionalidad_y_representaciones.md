# Reduccion de Dimensionalidad y Representaciones

**Nivel:** Licenciatura, semestre 5

**Bloque:** Machine learning clasico

**Proposito:** Reducir variables, visualizar datos de alta dimension y construir representaciones utiles.

**Prerrequisitos:** Algebra lineal, estadistica y Python.

## Temario por unidades

1. Motivacion: maldicion de dimensionalidad, ruido y visualizacion.
2. PCA: SVD, varianza explicada y reconstruccion.
3. Kernel PCA y metodos no lineales.
4. t-SNE y UMAP para visualizacion exploratoria.
5. Factor analysis e ICA.
6. Autoencoders introductorios.
7. Seleccion de variables vs extraccion de variables.

## Practicas en Python

- Aplicar PCA, t-SNE y UMAP.
- Visualizar embeddings y evaluar perdida de informacion.

## Conexion con industria en Mexico

Reducir variables en encuestas, sensores industriales, comportamiento de clientes, texto y fraude.

## Evaluacion sugerida

Notebook comparativo con interpretacion de componentes y riesgos.
