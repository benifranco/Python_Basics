# Cloud, DevOps y Plataformas de Datos

**Nivel:** Maestria profesional

**Bloque:** Ingenieria y produccion

**Proposito:** Entender infraestructura moderna para datos y ML sin depender de una sola nube.

**Prerrequisitos:** Terminal, Git, SQL y MLOps basico.

## Temario por unidades

1. Computo, almacenamiento, redes y seguridad en cloud.
2. IAM, secretos y buenas practicas de acceso.
3. Docker y fundamentos de Kubernetes.
4. Servicios administrados de datos y ML.
5. Infraestructura como codigo.
6. Cost management y FinOps.
7. Arquitecturas hibridas y restricciones regulatorias.

## Practicas en Python

- Desplegar una API y un job batch.
- Automatizar tareas con scripts y pipelines.

## Conexion con industria en Mexico

Considerar costos, residencia de datos, conectividad, equipos mixtos y adopcion gradual en empresas mexicanas.

## Evaluacion sugerida

Arquitectura cloud documentada con estimacion de costo y riesgos.

## Evidencias de portafolio

- Repositorio limpio con README, ambiente reproducible y datos documentados.
- Notebook o script principal con resultados verificables.
- Reporte tecnico breve y presentacion ejecutiva.
