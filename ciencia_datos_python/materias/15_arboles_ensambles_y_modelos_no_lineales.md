# Arboles, Ensambles y Modelos No Lineales

**Nivel:** Licenciatura, semestre 5

**Bloque:** Machine learning clasico

**Proposito:** Aprender modelos flexibles ampliamente usados en industria para datos tabulares.

**Prerrequisitos:** Clasificacion, regresion y validacion.

## Temario por unidades

1. Arboles de decision para regresion y clasificacion.
2. Impureza, ganancia, poda y profundidad.
3. Bagging y random forest.
4. Boosting: AdaBoost, gradient boosting, XGBoost, LightGBM y CatBoost.
5. Importancia de variables y dependencia parcial.
6. Monotonicidad, restricciones y explicabilidad.
7. Riesgo de sobreajuste y validacion.

## Practicas en Python

- Comparar arbol, random forest y boosting.
- Interpretar feature importance, SHAP basico y PDP.

## Conexion con industria en Mexico

Modelar riesgo, ventas, inventario, produccion, churn y deteccion de anomalias tabulares.

## Evaluacion sugerida

Proyecto tabular con validacion, explicacion y analisis de errores.

## Evidencias de portafolio

- Repositorio limpio con README, ambiente reproducible y datos documentados.
- Notebook o script principal con resultados verificables.
- Reporte tecnico breve y presentacion ejecutiva.
