# MLOps, Despliegue y Monitoreo de Modelos

**Nivel:** Maestria profesional

**Bloque:** Ingenieria y produccion

**Proposito:** Llevar modelos a produccion de forma confiable, medible y mantenible.

**Prerrequisitos:** ML supervisado, software y datos.

## Temario por unidades

1. Ciclo de vida ML: datos, features, entrenamiento, validacion, despliegue y monitoreo.
2. Empaquetado, APIs, contenedores y servicios.
3. Feature stores y versionado de modelos.
4. CI/CD para ML.
5. Monitoreo: performance, drift, calidad de datos y latencia.
6. Retraining, canary releases y rollback.
7. Seguridad, auditoria y trazabilidad.

## Practicas en Python

- Crear una API con FastAPI.
- Empaquetar un modelo con Docker.
- Registrar experimentos y metricas.

## Conexion con industria en Mexico

Implementar modelos en fintech, retail, manufactura, logistica, salud y gobierno con trazabilidad.

## Evaluacion sugerida

Servicio ML desplegable localmente con monitoreo basico.

## Evidencias de portafolio

- Repositorio limpio con README, ambiente reproducible y datos documentados.
- Notebook o script principal con resultados verificables.
- Reporte tecnico breve y presentacion ejecutiva.
