# Adquisicion, Limpieza y ETL de Datos

**Nivel:** Licenciatura, semestre 3

**Bloque:** Datos y analitica

**Proposito:** Aprender que la mayor parte del valor en ciencia de datos empieza con datos confiables y bien preparados.

**Prerrequisitos:** Python, pandas y SQL basico.

## Temario por unidades

1. Fuentes de datos: archivos, APIs, bases relacionales y scraping responsable.
2. Tipos de errores: duplicados, faltantes, outliers, unidades y codificacion.
3. Validacion de esquemas, reglas de negocio y perfiles de datos.
4. Transformaciones con pandas y SQL.
5. Pipelines ETL y ELT.
6. Versionado de datos y documentacion.
7. Introduccion a calidad, linaje y catalogos de datos.

## Practicas en Python

- Crear pipelines con pandas y funciones puras.
- Consumir APIs publicas.
- Generar reportes de calidad de datos.

## Conexion con industria en Mexico

Preparar datos abiertos de INEGI, Data Mexico, Banxico o portales estatales.

## Evaluacion sugerida

Pipeline reproducible con datos sucios y bitacora de decisiones.

## Evidencias de portafolio

- Repositorio limpio con README, ambiente reproducible y datos documentados.
- Notebook o script principal con resultados verificables.
- Reporte tecnico breve y presentacion ejecutiva.
