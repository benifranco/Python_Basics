# Big Data y Sistemas Distribuidos

**Nivel:** Maestria

**Bloque:** Ingenieria y produccion

**Proposito:** Procesar datos grandes con arquitecturas distribuidas y criterios de costo, confiabilidad y latencia.

**Prerrequisitos:** SQL, Python y fundamentos de sistemas.

## Temario por unidades

1. Arquitecturas batch, streaming y lambda/kappa.
2. Paralelismo, particionamiento y tolerancia a fallas.
3. Spark: DataFrames, SQL, MLlib y optimizacion.
4. Kafka y fundamentos de streaming.
5. Formatos columnares: Parquet, ORC y compresion.
6. Data lakes, lakehouse y catalogos.
7. Costos, rendimiento y gobierno de datos.

## Practicas en Python

- Procesar datos con PySpark.
- Crear un pipeline batch y uno streaming simulado.

## Conexion con industria en Mexico

Transacciones retail, telecom, sensores de manufactura, pagos digitales, logistica y datos gubernamentales.

## Evaluacion sugerida

Pipeline escalable con medicion de rendimiento y costo.

## Evidencias de portafolio

- Repositorio limpio con README, ambiente reproducible y datos documentados.
- Notebook o script principal con resultados verificables.
- Reporte tecnico breve y presentacion ejecutiva.
