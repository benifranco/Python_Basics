# Terminal, Git y Reproducibilidad

**Nivel:** Licenciatura, semestre 1

**Bloque:** Herramientas computacionales

**Proposito:** Aprender a trabajar como profesional: terminal, control de versiones, ambientes virtuales y estructura de proyectos.

**Prerrequisitos:** Programacion basica o cursarla en paralelo.

## Temario por unidades

1. Terminal: rutas, archivos, permisos, redireccion y pipes.
2. Git: init, add, commit, branch, merge, diff y log.
3. GitHub: repositorios, issues, pull requests y README.
4. Ambientes virtuales: venv, pip, requirements.txt y pyproject.toml.
5. Estructura de proyectos: src, tests, notebooks, data y docs.
6. Reproducibilidad: semillas, dependencias, versiones y notebooks limpios.

## Practicas en Python

- Crear un proyecto Python desde cero.
- Registrar dependencias y ejecutar pruebas.
- Publicar un repositorio con README tecnico.

## Conexion con industria en Mexico

Preparar proyectos que puedan compartirse con equipos academicos, gobierno o empresas sin depender de rutas personales.

## Evaluacion sugerida

Proyecto reproducible con ambiente virtual, pruebas y documentacion.

## Evidencias de portafolio

- Repositorio limpio con README, ambiente reproducible y datos documentados.
- Notebook o script principal con resultados verificables.
- Reporte tecnico breve y presentacion ejecutiva.
