# Algebra Lineal para Datos

**Nivel:** Licenciatura, semestre 2

**Bloque:** Fundamentos matematicos

**Proposito:** Dominar vectores, matrices y transformaciones como lenguaje central de datos, modelos y redes neuronales.

**Prerrequisitos:** Algebra y programacion basica.

## Temario por unidades

1. Vectores, matrices, normas, producto punto y proyecciones.
2. Sistemas lineales, rango, independencia y espacios vectoriales.
3. Bases, cambio de base y transformaciones lineales.
4. Ortogonalidad, minimos cuadrados y proyecciones.
5. Autovalores, autovectores y diagonalizacion.
6. Descomposiciones: LU, QR, SVD y PCA.
7. Matrices dispersas y computo numerico.

## Practicas en Python

- Resolver sistemas con NumPy.
- Implementar minimos cuadrados y PCA basico.
- Visualizar transformaciones en 2D.

## Conexion con industria en Mexico

Aplicar SVD/PCA a encuestas, sensores industriales y segmentacion de clientes.

## Evaluacion sugerida

Examen teorico, laboratorio de SVD/PCA y proyecto corto de reduccion de datos.

## Evidencias de portafolio

- Repositorio limpio con README, ambiente reproducible y datos documentados.
- Notebook o script principal con resultados verificables.
- Reporte tecnico breve y presentacion ejecutiva.
