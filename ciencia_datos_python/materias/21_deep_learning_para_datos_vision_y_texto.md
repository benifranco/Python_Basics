# Deep Learning para Datos, Vision y Texto

**Nivel:** Licenciatura avanzada / Maestria

**Bloque:** IA avanzada

**Proposito:** Comprender redes neuronales modernas y aplicarlas con criterio a problemas de vision, texto y datos estructurados.

**Prerrequisitos:** Algebra lineal, calculo, optimizacion y ML supervisado.

## Temario por unidades

1. Perceptron multicapa, backpropagation y funciones de activacion.
2. Regularizacion, dropout, batch normalization y data augmentation.
3. Optimizadores: SGD, Adam y schedules.
4. CNN para imagenes.
5. RNN, LSTM y atencion.
6. Transformers y modelos preentrenados.
7. Autoencoders y modelos generativos introductorios.
8. Practicas de entrenamiento: GPUs, experiment tracking y reproducibilidad.

## Practicas en Python

- Entrenar redes con PyTorch o TensorFlow.
- Usar transfer learning.
- Evaluar modelos en datos no balanceados.

## Conexion con industria en Mexico

Vision industrial, inspeccion de calidad, OCR de documentos, NLP en espanol, mantenimiento y salud.

## Evaluacion sugerida

Proyecto deep learning con baseline clasico y analisis de errores.

## Evidencias de portafolio

- Repositorio limpio con README, ambiente reproducible y datos documentados.
- Notebook o script principal con resultados verificables.
- Reporte tecnico breve y presentacion ejecutiva.
