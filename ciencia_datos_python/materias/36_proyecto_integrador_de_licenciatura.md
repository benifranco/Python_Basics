# Proyecto Integrador de Licenciatura

**Nivel:** Licenciatura, semestre 8

**Bloque:** Industria, etica y capstone

**Proposito:** Integrar datos, modelado, visualizacion, evaluacion y comunicacion en un proyecto completo.

**Prerrequisitos:** EDA, SQL, regresion, clasificacion y visualizacion.

## Temario por unidades

1. Definicion de problema y stakeholders.
2. Plan de datos, permisos y riesgos.
3. EDA, limpieza y baseline.
4. Modelo interpretable y modelo competitivo.
5. Validacion, analisis de error y limitaciones.
6. Producto minimo: dashboard, API o informe.
7. Presentacion final y defensa tecnica.

## Practicas en Python

- Proyecto reproducible en repositorio.
- Notebook ejecutivo, scripts y documentacion.

## Conexion con industria en Mexico

Resolver un caso de sector mexicano con fuentes publicas, simuladas o datos provistos por una organizacion.

## Evaluacion sugerida

Entrega escrita, repositorio, presentacion y demo.

## Evidencias de portafolio

- Repositorio limpio con README, ambiente reproducible y datos documentados.
- Notebook o script principal con resultados verificables.
- Reporte tecnico breve y presentacion ejecutiva.
