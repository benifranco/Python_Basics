# Bases de Datos SQL y Modelado Relacional

**Nivel:** Licenciatura, semestre 3

**Bloque:** Datos y analitica

**Proposito:** Extraer y modelar datos estructurados, una habilidad central para analistas, cientificos e ingenieros de datos.

**Prerrequisitos:** Programacion basica.

## Temario por unidades

1. Modelo relacional: tablas, llaves primarias y foraneas.
2. SELECT, WHERE, ORDER BY, GROUP BY y HAVING.
3. JOIN: inner, left, right, full y self joins.
4. Subconsultas, CTEs y ventanas.
5. Normalizacion, desnormalizacion y modelos estrella.
6. Indices, planes de consulta y nociones de rendimiento.
7. Transacciones, consistencia y calidad de datos.

## Practicas en Python

- Usar SQLite/PostgreSQL desde Python.
- Construir consultas para dashboards y modelos.
- Comparar SQL con pandas.

## Conexion con industria en Mexico

Resolver casos de ventas, facturacion, inventarios, credito, telecom y gobierno digital.

## Evaluacion sugerida

Examen practico de SQL y proyecto de base relacional.

## Evidencias de portafolio

- Repositorio limpio con README, ambiente reproducible y datos documentados.
- Notebook o script principal con resultados verificables.
- Reporte tecnico breve y presentacion ejecutiva.
