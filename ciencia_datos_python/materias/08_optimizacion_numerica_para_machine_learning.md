# Optimizacion Numerica para Machine Learning

**Nivel:** Licenciatura, semestre 3

**Bloque:** Fundamentos matematicos

**Proposito:** Entender como se minimizan funciones de perdida en modelos estadisticos y de aprendizaje automatico.

**Prerrequisitos:** Calculo, algebra lineal y Python.

## Temario por unidades

1. Funciones convexas y no convexas.
2. Gradiente descendente, momentum y metodos adaptativos.
3. Condiciones de optimalidad y multiplicadores de Lagrange.
4. Regularizacion L1 y L2 como penalizacion.
5. Optimizacion con restricciones.
6. Problemas numericos: escala, condicionamiento y convergencia.
7. Introduccion a optimizacion estocastica.

## Practicas en Python

- Implementar optimizadores desde cero.
- Comparar convergencia en regresion lineal y logistica.

## Conexion con industria en Mexico

Optimizar precios, inventarios, rutas, asignacion de personal y parametros de modelos.

## Evaluacion sugerida

Laboratorios y proyecto de optimizacion aplicada.

## Evidencias de portafolio

- Repositorio limpio con README, ambiente reproducible y datos documentados.
- Notebook o script principal con resultados verificables.
- Reporte tecnico breve y presentacion ejecutiva.
