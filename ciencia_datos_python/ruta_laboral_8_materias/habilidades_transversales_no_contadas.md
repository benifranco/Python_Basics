# Habilidades transversales no contadas como materia

Estas habilidades no estan dentro de las 8 materias elegidas, pero deben practicarse durante toda la ruta. Sin ellas, el conocimiento tecnico se ve incompleto en entrevistas y proyectos.

## 1. Terminal, Git y GitHub

Minimo esperado:

- usar `cd`, `ls`, `pwd`, `mkdir`, `python script.py`,
- crear y activar ambientes virtuales,
- usar `git init`, `git add`, `git commit`, `git status`, `git log`,
- subir proyectos a GitHub,
- escribir README claro,
- evitar subir `.env`, datos sensibles o credenciales.

## 2. Excel o Google Sheets

Aunque la ruta sea en Python, muchos equipos en Mexico siguen usando hojas de calculo.

Minimo esperado:

- tablas dinamicas,
- filtros,
- formulas basicas,
- limpieza rapida,
- validacion manual de resultados,
- exportacion/importacion CSV.

## 3. Una herramienta BI

Elige una:

- Power BI,
- Tableau,
- Looker Studio,
- Metabase,
- Streamlit si el curso quiere mantenerse en Python.

Minimo esperado:

- conectar datos,
- crear KPIs,
- filtros,
- graficas,
- dashboard de una pagina,
- explicar hallazgos en lenguaje ejecutivo.

## 4. Comunicacion ejecutiva

Minimo esperado:

- explicar el problema en una frase,
- separar hallazgos de opiniones,
- declarar supuestos,
- decir que no se puede concluir,
- proponer una accion,
- hablar de costo de error,
- cerrar con siguientes pasos.

## 5. Ingles tecnico

Minimo esperado:

- leer documentacion,
- buscar errores,
- entender mensajes de librerias,
- leer tutoriales y articulos tecnicos,
- escribir nombres de variables claros.

## 6. Entrevistas tecnicas

Practicar:

- SQL en pizarra o editor,
- pandas basico,
- interpretacion de graficas,
- metricas de clasificacion,
- explicacion de proyectos propios,
- preguntas de negocio: "que harias si el modelo falla?".

## 7. Etica y privacidad

Minimo esperado:

- anonimizar datos sensibles,
- no usar informacion personal sin permiso,
- documentar sesgos,
- explicar limites,
- no automatizar decisiones sensibles sin supervision humana.
