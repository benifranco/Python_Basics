# Visualizacion y Comunicacion de Datos

**Nivel:** Licenciatura, semestre 3

**Bloque:** Datos y analitica

**Proposito:** Traducir datos en explicaciones claras para exploracion, analisis y comunicacion ejecutiva.

**Prerrequisitos:** Estadistica descriptiva y pandas.

## Temario por unidades

1. Principios de percepcion visual y seleccion de graficas.
2. Histogramas, boxplots, scatter plots, lineas, barras y mapas.
3. Visualizacion de incertidumbre.
4. Dashboards: objetivos, metricas y filtros.
5. Narrativa con datos y comunicacion ejecutiva.
6. Errores comunes: escalas truncadas, sobrecarga visual y correlaciones falsas.
7. Accesibilidad, color y diseno responsable.

## Practicas en Python

- Usar matplotlib, seaborn y plotly.
- Construir un dashboard simple.
- Preparar un informe reproducible.

## Conexion con industria en Mexico

Comunicar indicadores de ventas, empleo, salud, manufactura o seguridad publica a audiencias no tecnicas.

## Evaluacion sugerida

Portafolio de visualizaciones y presentacion ejecutiva.

## Evidencias de portafolio

- Repositorio limpio con README, ambiente reproducible y datos documentados.
- Notebook o script principal con resultados verificables.
- Reporte tecnico breve y presentacion ejecutiva.
