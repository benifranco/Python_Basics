# Ejercicios

En esta carpeta los ejercicios se organizan por tema en subcarpetas. Ejemplos:
- `01_intro/`
- `02_strings/`
- `03_listas_y_diccionarios/`

Cada subcarpeta incluye:
- `enunciado.md` con las instrucciones.
- Archivos iniciales (`main.py`, datos de apoyo) que los alumnos completan.
- Pruebas opcionales `test_*.py` pequenas que validan la solucion.
