# Documentacion del curso

En esta carpeta se muestran guias, apuntes y enlaces de referencia.

## Ideas de contenido
- Instrucciones de instalacion para diferentes sistemas operativos.
- Resumen de sintaxis basica de Python.
- Temario y calendario de sesiones.
- Glosario de terminos y buenas practicas.

## Temario inicial (10 sesiones)
1. Introduccion y entorno: instalacion, uso de la terminal, `venv`, pip, primer script `hola_mundo.py`.
2. Fundamentos del lenguaje: tipos primitivos, input/output, conversiones y errores comunes.
3. Control de flujo: condicionales, operadores logicos, expresiones booleanas.
4. Ciclos y colecciones: `for`, `while`, listas, tuplas y slicing.
5. Strings a fondo: metodos utiles, formato con f-strings, validaciones sencillas.
6. Diccionarios y conjuntos: operaciones basicas, iteracion, patrones de uso.
7. Funciones: definicion, argumentos, retorno, docstrings y manejo de errores con `try/except`.
8. Modulos y paquetes: organizar codigo en `src/`, imports, `__init__.py`, uso de bibliotecas estandar.
9. Archivos y datos: leer/escribir texto y CSV, rutas con `pathlib`, buenas practicas de IO.
10. Pruebas y calidad: introduccion a `pytest`, asserts, casos basicos y estructura en `tests/`.
